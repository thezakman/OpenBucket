## Usage
<pre>
usage: OpenBucket.py bucket_url

positional arguments:
  bucket_url               Url of the bucket to download
</pre>
 
## Installation
  1. (Optional) `virtualenv venv && source ./venv/bin/activate`
  2. `pip install -r requirements.txt`
